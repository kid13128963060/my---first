Extract all frames

 

 Top  Previous  Next

 

 

Click on the Options Menu, then Extract all frames. A dialog allows you to select the folder/directory in which the frames will be stored. Find the folder/directory you want, then click OK to have the images stored there. This option is available for animated GIFs, animated cursors (ANI) and supported video files.

 

The frames from the image are placed into the folder/directory with names like this:

 

 imagename_frame_001.bmp

 

where imagename is the name of the original image and 001 is the number of the frame.

 

To exit without extracting the frames, click Cancel.
