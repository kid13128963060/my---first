Browsing

 

View other files in folder: Click this to have the viewer display in each folder/directory other files with extensions supported by IrfanView. If this is unchecked, you can't open next/previous file with space/backspace or with the toolbar arrows. Hint: the list of scanned extensions can be set/changed in "Properties->Extensions->Load custom file types". You can delete unwanted extensions, press OK and later F5 in IrfanView to refresh/rescan the folder.

 

Show hidden files/folders during moving through folder: Enable/disable to show hidden files in the actual folder/directory.

 

Show browse buttons on left/right side of the main window: This can show arrow buttons in the main window to load next/previous file from the file list.

 

If the end/begin of the folder is reached, (during browsing). You can choose what will happen when you reach the last/first of the images in a folder/directory: IrfanView can show you a special browse folder dialog to use another folder/subfolder; or you can loop the current folder and just go on to the first/last image again; or nothing will happen, the browsing will stop.

 

Beep on folder loop or Screenshot save: get some info if something like this is happened in background.

 

Load next/previous file on Mouse wheel: you can disable mouse wheel browsing for images without scrollbar.

 

Jump always to next image if Page keys or Mouse wheel used: Don't scroll large images, jump always to previous/next image, if you press Page keys or roll the mouse wheel.

 

 

Editing

 

Undo option: you can set the number of Undo/Redo steps. Maximal number is 20, 0 will disable the Undo/Redo feature.

 

Set selection/grid color: you can set the color (actually: the inverted color) of the selection rectangle or grid cells. By default, the rectangle color is gray.

 

Import palette: by default, the image colors will be changed 1:1 to colors in the PAL file, this options will search for the best/nearest color in the PAL file.

 

There is also an option to set the background color which is used for the cut function.

 

Paste into selection: by default, the pasted image will be stretched to selection, this option allows you to paste the image proportionally.

 

Paste Special: by default, the pasted image will be fitted to the existing image, this option allows you to paste the image in original size.

 

Selection border cell size: you can set the thickness of selection rectangle lines.

 

Set grid cell size: you can set the size of the grid cells (edit menu).

 

Tolerance value for the edit menu "Auto crop borders": you can set a value to ignore some (similar) border pixels and crop a larger border.

 

Name for pasted image: you can use placeholders to set a custom filename (instead of "Clipboard01", "Clipboard02" ...) for the pasted image. The default name is "Clipboard_$U(%m-%d-%Y)_##", which also adds a timestamp to the filename text ('##' is the counter, 2 digits).

 

You can set the Options for TXT-files: font, font color (text) and background color.
