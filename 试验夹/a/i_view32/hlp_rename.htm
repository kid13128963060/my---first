Rename File

 

 Top  Previous  Next

 

 

Click on the File Menu, select Rename File. A dialog opens; type in a new name for the file and click OK. The filename will be changed, both in the viewer and on the hard drive (or other media). To exit without making a change, click Cancel.

 

If Rename sidecar files is checked, the program will also rename all other files with same (old) base name and different extension.

"Sidecar" files have same base name but different extension(s). An example, there is a file:

image001.jpg

and there are sidecar files like:

image001.cr2

image001.txt

 

If you rename "image001.jpg" to e.g. "my city 001.jpg" and enable the sidecar-option, the sidecar files will be renamed to:

my city 001.cr2

my city 001.txt

 

 

Note: the dialog will remember last position. If the rename is not possible (e.g. permission problems or the filename already exists), the dialog will raise again.
