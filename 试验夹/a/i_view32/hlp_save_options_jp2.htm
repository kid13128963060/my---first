JPEG2000

 

Click one of the option buttons to set a compression level for JP2 images. If you select Quality or Compress to bytes, type in a value for the option. Lossless compression allows saving of the image without loosing any image information (= highest quality).

 

Note:Saving as JPEG2000 is only available, if the PlugIn JPEG2000 is installed.

 

 

After setting the options you choose, click Save to save the image and store the settings. To exit without saving the file or changed options, click Cancel.

 

The new settings will stay in effect until you change them again.
