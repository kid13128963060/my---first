Hotfolder (watch folder)

 

 Top  Previous  Next

 

 

Click on the Options Menu, then Hotfolder. A dialog shows you some options.

 

You can set which folder should be watched. All newly added images/files in that folder will be displayed in the IrfanView window.

The window can be cleared if you set the clear-display delay > 0.

 

When you click on the Start button, the main program window will minimize and you see the "Watching folder" text in the window/taskbar title.

The program is waiting for new files in the watch folder (and subfolders, if activated).

 

To stop watching you can close IrfanView or open the Hotfolder dialog again (and press Cancel).
