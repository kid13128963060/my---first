Next/Previous File in Folder/List

 

 Top  Previous  Next

 

 

Click on the View Menu, then click Next File in Folder/List (or use the Next File tool on the toolbar). The viewer displays the next supported file.

 

You can also use the Space key to move to the next file in the folder/list. If the image does not have a vertical scrollbar at its right edge or you have enabled the appropriate property option at Browsing/Editing, you can also use the Page Down key or roll the mouse wheel down to move to the next file in the folder/list.

 

----

 

Click on the View Menu, then click Previous File in Folder/List (or use the Previous File tool on the toolbar). The viewer displays the previous supported file, in reverse sequence.

 

You can also use the Backspace key to move to the previous file in the folder/list. If the image does not have a vertical scrollbar at its right edge or you have enabled the appropriate property option at Browsing/Editing, you can also use the Page Up key or roll the mouse wheel up to move to the previous file in the folder/list.

 

 

Note:The files are sorted as set in menu Options->Sort list files.
