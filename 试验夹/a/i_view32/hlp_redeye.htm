Red Eye Reduction

 

 Top  Previous  Next

 

 

Click on the Image Menu and click Red eye reduction.

 

This menu can only be used when you have created a selection/outline in the image using the left mouse button. The feature reduces the red color values in the selected area (usually the eye). The amount can be changed using Effects Setup.

 

The Effects Setup dialog allows also a Green or Yellow Eye Reduction for cats/dogs etc.

 

This action can be reversed with the Edit Menu's Undo command.
