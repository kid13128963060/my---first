Sort file list

 

 Top  Previous  Next

 

 

Click on the Options Menu and point at Sort file list. A sub-menu appears with the choices by Name, by Name (XP style), by Date, by Size and by Extension. These are available in either ascending (default) or descending order. The final choice is No sort. Click your choice.

 

The sort option by Name differs from by Name (natural) by a different interpretation of numbers in filenames. With by Name File10.jpg is sorted before File1.jpg. With Name (natural) the sort order is File1.jpg before File10.jpg.

 

When you activate any of the options that use the space bar to move forward and backspace to move backward through the files, such as Full Screen mode or Thumbnails, they will be in the order set by your sort option.

 

Hint:In the Thumbnail window, you have additional sort options like: Sort by Image Width/Height or manual sort using Drag & Drop etc.
