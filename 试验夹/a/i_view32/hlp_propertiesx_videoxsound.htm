Properties, Video/Sound

 

 Top  Previous  Next

 

 

This dialog shows you options for video/sound playing. Videos/sounds can be played in the main window (default) or with the English IV_Player PlugIn in a separate window. For playing video/sound also the PlugIn Video must be installed, which is done during program installation.

 

Use DirectShow for playing: this option can play (with installed codecs!) most video formats.

If activated, you can use the Pause key for pause/resume and left/right arrow keys to seek in the video, see Hotkeys page.

The player window offers some additional options (zoom, volume, copy frame) => show the context menu for infos (the click with the right mouse button).

 

Try to play FLV files in main window: this option can play (with installed codecs!) FLV files in the main window instead of the extra Flash Player window.

 

Apple QuickTime options:

IrfanView has QuickTime support! This allows IrfanView to read following formats: MOV, QTIF, Mac PICT, FLI/FLC. You can enable/disable this support for these formats. To play QuickTime files with IrfanView, the Quicktime PlugIn and Apple QuickTime must be installed.

 

Hint: A good codec pack is the K-Lite Codec Pack, it contains the most popular video/audio codecs.
