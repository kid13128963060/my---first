How to start IrfanView slideshow from a CD/DVD?

 

 Top  Previous  Next

 

 

1. Method:The easiest way is, if you have Windows XP (or later) software installed:

 

1.1.Create and test (play) the slideshow in the Slideshow dialog.

 

1.2.Press in the slideshow dialog the button Burn to CD/DVD.

 

1.3.Choose the burn option, Data-CD/DVD is default/recommended.

 

1.4.Insert empty CD/DVD-R/-RW in your burning device and burn it.

 

1.5.Done.

 

 

 

2. Method:Manual or special slideshow CD/DVD:

 

2.1.Copy IrfanView files into the root CD/DVD folder. (You can also copy an INI file "i_view32.ini" with your settings.)

 

2.2.Copy your images into the CD/DVD folder called e.g. "images".

 

2.3.Create a TXT file (in slideshow dialog or using Notepad) with the names of the images. If your image files have names like "image1.jpg", "image2.jpg", etc. the TXT should look like this:

 

.\images\image1.jpg

.\images\image2.jpg

etc.

 

In each line is a filename with relative paths (relative to IrfanView, i_view32.exe).

 

Comment text: if the first line character is a ";", the line/text will be ignored (can be used as comment).

 

Save the TXT file e.g. as "slideshow.txt" in the CD/DVD root folder.

 

2.4.Create the "autorun.inf" file (Notepad). Write in the file 2 lines:

 

[AutoRun]

open = i_view32.exe /slideshow=slideshow.txt

 

Save "autorun.inf".

 

2.5.Done. Burn the CD/DVD using your burning software.
