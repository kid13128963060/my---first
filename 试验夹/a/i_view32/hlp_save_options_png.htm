PNG

 

If the image contains transparency (color or alpha layer), it will be automatically used for saving.

Set the compression level and transparent color (similar to GIF/ICO) for PNG images. Note: PNG is a lossless format (like GIF) with true color support. It is usually the ideal format for screenshots, because the file size for this images is often much smaller as i.e. JPG, the saved image has at the same time no loss in quality and can be read by many programs.

 

Info:If the image contains transparency (color or alpha layer), it will be automatically used for saving, no need to set a custom transparency color.

 

Save with a Custom transparent color: after you click on the Save button, a preview dialog will be shown to click into the preview image to set a new/custom transparent color. If you activate the option Use main window color for transparency, the current window color (Properties->Viewing) will be automatically set as transparent color.

Save transparency as alpha channel: this option will generate a "synthetic" alpha layer based on the image colors and window background color. The result is not always perfect.

 

You can also try the OptiPNG PlugIn for optimized PNG saving. It can reduce the result file size but it needs more time.

 

 

PBM, PGM, PPM

 

Click one of the option buttons to set an encoding type for PBM/PGM/PPMs. Ascii will produce very large files.

 

 

ICO

 

Info:If the image contains transparency (color or alpha layer), it will be automatically used for saving, no need to set a custom transparency color.

 

You can set a Custom transparency color during saving with the click on the preview image (will be shown later).

If you activate the option Use main window color for transparency, the current window color (Properties->Viewing) will be automatically set as transparent color.

 

 

After setting the options you choose, click Save to save the image and store the settings. To exit without saving the file or changed options, click Cancel.

 

The new settings will stay in effect until you change them again.
