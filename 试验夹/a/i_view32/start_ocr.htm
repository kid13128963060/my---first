http://www.irfanview.info/plugins/kadmos/

 

 

Hint:The option can be used on a smaller portion (rectangle selection) of an image as well as on the entire image. If you want to use a smaller portion of the image, use mouse to select an area and then call the OCR dialog.

 

 

Click on the Options Menu, then Start OCR. A window is opened which allows you to use OCR (Optical Character Recognition) to acquire text from image pixels and copy it to clipboard or save as file.
