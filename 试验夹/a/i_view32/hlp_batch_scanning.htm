Acquire/Batch scanning

 

 Top  Previous  Next

 

 

Click on the File Menu, click Acquire/Batch scanning.

 

·If you want to acquire just one image, use the first option.

 

The program that runs your scanner or any other TWAIN compatible device (digital camera, TV card, etc.) will appear. Follow the steps given in your scanner's manual for scanning an image.

 

When finished, the image will be placed into IrfanView.

 

 

·For many images, please use the second option.

 

There you can set:

 

•output filename
•starting index/counter for output files
•index increment number (can be also a negative number)
•number of index digits used in the filename
•option to skip existing files with same filename (no overwrite)
•remember last scan counter (will be set as start)
•destination folder/directory for scanned images
•image file type

 

Hint:  you can use some placeholders for the output filename. Allowed placeholders for the output filename: $W, $H, $B, $p and $U.

 Example: a filename text like: $U(%d_%m_%Y)-

 will produce result files like: 25_07_2013-01, 25_07_2013-02 etc.

 

 

If you are using the TIF or PDF format for output, you can create a multipage TIF/PDF file from the scanned images (only one file is saved as result, no increment).

 

 

The program that runs your scanner or any other TWAIN compatible device (digital camera, TV card, etc.) will appear. Follow the steps given in your scanner's manual for scanning an image. The TWAIN preview dialog won't be closed by IrfanView.

 

Images will be saved in the output folder/directory as image files. The filenames are "ScanImage001.bmp", "ScanImage002.bmp", etc. (if default settings used).

 

When finished, the last scanned image will be shown in the IrfanView window.
