Start Multimedia Player

Note:The Multimedia Player is only available with English PlugIn IV_Player, download and install:

 

http://www.irfanview.com/plugins.htm

 

 

Click on the Options Menu, then Start Multimedia Player.

 

The Multimedia Player allows IrfanView to play video, sound and audio-CD files like:

 

AVI, ASF, CDA, MP3, MPG/MPEG, MOV, MID/RMI, WAV, WMA , WMV, etc.

 

The player can play these formats through the MCI interface or using the Microsoft Media Player (recommended).

 

The player can extract frames from AVI files and has a built-in audio CD player.

 

You can close the Player with ESC key.
