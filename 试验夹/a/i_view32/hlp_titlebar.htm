Title Bar/Caption

 

 Top  Previous  Next

 

 

The title bar shows some informations about the displayed image at the top side of the main window.

 

These informations are:

 

·filename of the opened file (with full path or only filename)
·Pixel coordinates, color value and palette index - when you click (and the mouse) into the image
·Selection rectangle coordinates, dimensions and aspect ratio - if a selection rectangle is visible
·Dimensions of the zoomed image - if zoom activated

 

Turn the title bar on by using the View Menu and selecting the Show/hide caption option. Turn the bar off by again clicking on View and Show/hide caption.
