TIFF

 

Click one of the option buttons to set a compression type for TIFFs. The left side is for color images, the right may be used for black & white.

 

Save palette for grayscale images: some graphic editors have "problems" with grayscale TIFs with a palette. So you can disable the palette.

 

If you have a multipage image opened, you can save all pages from the input image into the result TIF, when you activate the option: Save all pages from original image.

 

 

After setting the options you choose, click Save to save the image and store the settings. To exit without saving the file or changed options, click Cancel.

 

The new settings will stay in effect until you change them again.
