Increase/Decrease Color Depth

 

 Top  Previous  Next

 

 

Click on the Image Menu, then select Increase Color Depth. A dialog shows the possible color depths. Select one.

 

Click on the Image Menu, then select Decrease Color Depth. A dialog shows the possible color depths. Select one.

Use best color quality: this option uses another (slower) method of color reduction. The result image will have a very high color accuracy comparing to the original image.

 

The dialog will try to set the last used BPP value (if possible for the current image).

 

Click OK to make the change.

 

 

This action can be reversed by the Edit Menu's Undo command.
