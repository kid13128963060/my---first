Print

 

 Top  Previous  Next

 

 

On the File Menu, click Print.

 

A dialog appears with options for printing an image.

 

You can select the printer and the orientation on the left dialog side.

 

A preview shows how the image would look printed in Portrait or Landscape orientation.

 

Use the Printer setup button to change the paper size or other printer settings.

 

 

Option buttons select how the image will print.

 

Original size will print the image with original dimensions, calculated from image DPI. DPI of 72 is used for images without DPI information.

Note:        If the image is larger than paper size, only the visible part will be printed.

 

Best fit to page puts the image on the page as large as it can, without altering the dimensions/ratio.

Note:Aspect ratio IS used.

 

Stretch to page fills the print area, even if it has to resize one dimension.

Note:NO aspect ratio is used.

 

Custom allows you to print the image with specific size. Type in the desired values for width, height and left/top margin in cm or inches.

Note:The maximal image size is limited by the paper size, aspect ratio is used when activated. The width/height values are not auto-calculated for a new image, check/uncheck the Aspect ratio option or click in the desired field.

 

Scale fits the image on the page. Type in the desired change; 1.00 is original size.

Note:The maximal image size is limited by the paper size.

 

 

Click Center image to have the image printed in the center of the paper. You can center the image in several ways: vertical or horizontal or both.

The X/Y position on paper values can be also negative numbers.

The option borderless printing will extend the image over the paper's un-printable area (depends on the printer). This option can be very useful if you print scanned papers. The option No overflow on page will reduce the image width/height if they exceed the paper size.

 

You can save/load profiles with values. A profile is a set of all dialog options. So you can have different profiles for different images and situations, no needs to change all options manually.

This feature is available in several dialogs.

 

You can print the filename of the image or its path or an user defined text as head- and/or footnote.

 

The options for multipage images are enabled only if you have a multipage file loaded, like a TIF file with several pages.

The option Print specific pages uses the ';' character as delimiter for page numbers. You can also use "-" for page range, like: 3-6; to print pages: 3, 4 , 5 and 6.

The option Reverse pages will reverse the pages order if a page range is used.

 

You can also set the Number of copies.

You can Collate (=sort) prints if a multipage image is loaded and the number of copies is bigger than 1.

 

Note:IrfanView saves some printer driver values and loads/sets them again on dialog start: Paper orientation, size, source, Color mode, Duplex mode.

 

Note:The preview image doesn't show the paper's un-printable area (depends on the printer).

 

 

Hint: If you have an outline area selection (rectangle) in the original image (IrfanView window), only the selected image area is printed (shown in the dialog).

 

 

Select options, then click Print.

 

Click Cancel to exit the Print dialog without printing, or when the job is done.
