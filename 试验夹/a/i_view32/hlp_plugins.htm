PlugIns

Please use always the current IrfanView and PlugIn version.

 

 

How to install IrfanView PlugIns?

 

1.Visit IrfanView's homepage: http://www.irfanview.com (or a mirror site).
2.Go to 'PlugIns' section.
3.Download PlugIns.
4.Click on the PlugIn file.
5.PlugIns will be installed into IrfanView 'PlugIns' folder/directory.

 

Note:You can enable/disable installed PlugIns in menu Help->Installed PlugIns.

 

 

Download:

 

You can download all PlugIns from this site:

 

 http://www.irfanview.com/plugins.htm

 

 

Please read also the file "i_plugins.txt" (IrfanView folder) for additional informations about the PlugIns.

 

Available PlugIns:

 

 

- ALTALUX:allows IrfanView to use AltaLux image effect.

 

- AWD:allows IrfanView to read Artweaver files.

 

- B3D:allows IrfanView to read BodyPaint 3D files.

 

- BABACAD4IMAGE:allows IrfanView to read DXF and DWG CAD files.

 

- BURNING:allows IrfanView to burn slideshow to a CD.

 Windows XP or later required, Nero option can burn Data DVD and VCD.

 

- CAMRAW:allows IrfanView to read Digital Camera RAW files.

 (ARW CR3 CR2 CRW DCR DNG ERF KDC MRW NEF NRW ORF PEF RAF RAW RW2 RWL SRF SRW X3F MOS)

 

- CADIMAGE:allows IrfanView to read CAD files (DXF, DWG, HPGL, CGM, SVG). Shareware, third party PlugIn.
Note:Check "i_plugins.txt" for details and newest version link.

 

- CRW:allows IrfanView to read Canon CRW/CR2 files (high resolution image version).

 Note:        This PlugIn requires additional Canon DLLs (check "i_plugins.txt" for details).

 

- DICOM:allows IrfanView to read Dicom medical files (DCM, ACR, IMA).

 

- DJVU:allows IrfanView to read AT&T DJVU files.

 

- DPX:allows IrfanView to read DPX/CIN (Digital Picture Exchange/Cineon) files.

 

- ECW:allows IrfanView to read/write ECW files.

 

- EFFECTS:contains image effects.

 

- EMAIL:allows IrfanView to send images as EMails.

 

- EXR:allows IrfanView to read EXR files.

 

- FACEDETECT:allows IrfanView to use the Face Detection feature from Thumbnails window.

 

- Film Simulation:allows IrfanView to change the picture colors using a CLUT file (Color-Lookup-Table).

 

- Filter Factory:allows IrfanView to use Filter Factory 8BF files (Photoshop PlugIns for Filter Factory).

 

- FLASH:allows IrfanView to read Macromedia Flash/Shockwave files.

 

- FLIF:allows IrfanView to read FLIF (Free Lossless Image Format) files.

 

- FORMATS:allows IrfanView to read some rare image formats.

 Current formats:        PCX, PSP, CLP, IFF/LBM, G3, RAS/SUN, RAW, Bio-RAD, Mosaic, XBM, XPM, GEM-IMG, WBMP, SGI, RLE, TTF, DDS, CAM, SFW, WAD, WAL, FITS, HDR, PVR, SIF, ANI

 Many old formats from Amiga, Atari, C64, ZX Spectrum etc.

 

- FPX:allows IrfanView to read FlashPix files.

 

- FTP:allows IrfanView transfer files from Thumbnails window using FTP.

 

- HDP:allows IrfanView to read HDP/WDP files (Microsoft HD Photo).

 

- ICONS:additional icons for IrfanView file associations.

 

- ICS:allows IrfanView to read ICS/IDS files.

 

- IMPDF:allows IrfanView (32-bit) to save PDF files.

 

- IMPDN:allows IrfanView to read PDN files.

 

- IrfanView Sandbox:allows IrfanView to use JewelScript Effects.

 

- IV_Player:allows IrfanView to play video/sound/audio-cd files.

 Current formats:        AVI, MPG, WAV, MIDI, CDA, WMA, ASF, AIF, SND, WMA/WMV, etc.

 

- JPEG2000:allows IrfanView to read/write JPEG 2000 files.

 

- JPEG_LS:allows IrfanView to read/write JLS files (JPEG-LS, JPEG Lossless).

 

- JPG Transformation:allows IrfanView to support lossless JPG transformations.

 

- JPM:allows IrfanView to read/write JPM files.

 

- KDC:allows IrfanView to read Kodak Digital Camera files.

 

- LCMS:allows IrfanView to use embedded color profiles in JPG/TIF files.

 

- MED:allows IrfanView (32-bit) to play MED/OctaMED audio files.

 

- METADATA:allows IrfanView to handle EXIF/IPTC/Comment information from compatible files.

 

- MNG:allows IrfanView to read/play MNG/JNG files.

 

- MP3:allows IrfanView to play MP3/MP2/MP1 files.

 

- MRC:allows IrfanView (32-bit) to read MRC files.

 

- MrSID:allows IrfanView to read LizardTech's SID files.

 NOTE: this PlugIn needs additional DLLs, download/install from:

 http://www.irfanview.net/plugins/irfanview_mrsid_plugin.exe

 or

 http://www.irfanview.net/plugins/irfanview_mrsid_plugin.zip

 

- NERO:allows IrfanView (32-bit) to burn slideshow to data or Video CD, using Nero Burning ROM software.

 Check http://www.nero.com for download.

 

- OCR_KADMOS:adds OCR features to IrfanView.

 Check http://www.irfanview.info/plugins/kadmos/ for download.

 

- PAINT:allows IrfanView to paint lines, circles, arrows, straighten image etc.

 

- PDF:allows IrfanView to to read/save PDF files.

 

- PhotoCD:allows IrfanView to read Kodak Photo CD files.

 

- PSHOST:allows IrfanView to load Adobe Photoshop 8BF filters.

 NOTE: some nice Adobe 8BF Plugins/effects can be downloaded/installed from:

 http://www.irfanview.net/plugins/irfanview_adobe_8bf_plugins.exe

 or

 http://www.irfanview.net/plugins/irfanview_adobe_8bf_plugins.zip

 Note:        some older/special 8BFs may require additional system DLLs (check "i_plugins.txt" for details).

 

- OptiPNG:allows optimized PNG saving.

 

- POSTSCRIPT:allows IrfanView to read PS/EPS/PDF/AI files.

 NOTE: Ghostscript is required: https://github.com/ArtifexSoftware/ghostpdl-downloads/releases/

 

- QuickTime:allows IrfanView to play Apple QuickTime movies. (MOV, FLI/FLC)

 

- RA_Player:allows IrfanView (32-bit) to play RealAudio RA files.

 

- REGIONCAPTURE:allows IrfanView to capture a rectangular area of the screen.

 

- SFF:allows IrfanView to read SFF (Structured Fax) files.

 

- Slideshow:allows IrfanView to create presentations in EXE or SCR format (stand-alone executable/screen saver).

 

- SoundPlayer:allows IrfanView to play OGG audio files (Ogg Vorbis).

 

- STUB_PLUGIN:allows IrfanView 64 bit to load some very old (32 bit) PlugIns.

 

- SVG:allows IrfanView to read SVG (Scalable Vector Graphic) files.

 

- TOOLS:contains some additional functions for IrfanView.

 

- VIDEO:allows IrfanView to play many video/sound formats and extract video frames.

 

- VIDEOEXPORT:allows IrfanView to save slideshow as MP4 video.

 

- WBZ:allows IrfanView to read WebShots (WBZ/WBC/WB1) files.

 

- WEBP:allows IrfanView to read/save Weppy (WEBP) files.

 

- WSQ:allows IrfanView to read WSQ (Wavelet Scaler Quantization) files.

 

- XCF:allows IrfanView to read XCF files.
