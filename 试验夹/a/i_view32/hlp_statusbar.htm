Status Bar

 

 Top  Previous  Next

 

 

The status bar shows some informations about the displayed image at the bottom side of the main window. These informations are described in the Image->Information dialog.

 

Turn the status bar on by using the View Menu, and selecting the Show/hide status bar option. Turn the bar off by again clicking on View and Show/hide status bar.

 

You can set the text shown in the field on the right.

 

If an image/file is tagged (hotkey: X), there will a "(X)" shown at the end of the field on the right.
