Copy/Move File

 

 Top  Previous  Next

 

 

Click on the File Menu, then Copy or Move File. The file in the viewer will be copied (or moved) to another folder/directory on your hard drive (or other media).

 

You can set a default "copy-to"/"move to" folder for Copy/Move at the Properties and copy/move all files to same folder or you can activate the option Ask me before copying/moving to show a dialog with several destinations for each file.

 

The Copy/Move File dialog offers many input fields for the definition of destination directories.

 

If File-Copy is used, you can set the option to create just a small shorcut file (LNK) to the original file in the destination folder. Existing LNK files (for same filename) will be overwritten.

 

 

Hint: you can use also relative paths, if you e.g. want to copy/move the file into a subfolder of the current working/file folder.

Example: your current file is "C:\images\image1.jpg". If you set as destination path (without "") like: ".\test" (the leading dot is very important!), the file will be copied (or moved) to "C.\images\test\image1.jpg". You can also write a more deep destination path like ".\test\big images\" etc. and the program will create/handle the whole path automatically.
