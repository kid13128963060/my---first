Display options

 

 Top  Previous  Next

 

 

Note:Only one of these options is active. Please remember when you change a display option and set it back in case of "problems"!

 

 

Fit window to image

The window will be resized to fit the image. Use this option if you want to have a 1:1 view of the image, no zoom.

 

Fit images to window

The image will be resized to fit the window.

 

Fit only big images to window

Click this to have the viewer reduce oversized images until they can be seen fully in the window. Images smaller than the window will not be changed.

 

Fit images to window width/height

The image will be resized to fit the window width/height.

 

Fit small image side to window

The image will be resized to fit the window with the smaller image side, the larger side will be proportionally adjusted.

 

Fit images to desktop

The window/image will be resized proportional to fit the desktop size.

 

Fit only big images to desktop (this option has been set as default option)

Click this to have the viewer reduce oversized images until they can be seen fully on the desktop. Images smaller than the desktop will not be changed.

 

Fit images to desktop width

The window/image will be resized to fit the desktop width.

 

Fit images to desktop height

The window/image will be resized to fit the desktop height.

 

Fit small image side to desktop

The image will be resized to fit the desktop with the smaller image side, the larger side will be proportionally adjusted.

 

Do not fit anything

No change in window size/position if the user loads a new image.

 

 

Use "Resample" for fitting

This option will improve the display quality for fitted images (fit to window or desktop).

 

Use "Resample" for zooming

This option will improve the display quality for zoomed images.

 

Apply sharpen after "Resample"

The Resample option adds usually some bluriness to the displayed image. This option will call the Sharpen menu/function after resampling to reduce it.

 

 

Center image in window

The image can be shown centered in the window or in the top-left window corner.
