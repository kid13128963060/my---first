Edit or set alpha/transparency color

 

 Top  Previous  Next

 

 

Click on the Image Menu, then Edit or set alpha/transparency color. A dialog appears with a slider control and buttons.

 

With CTRL + click into the image (main window) you can set a (new) transparency color.

 

The slider changes the intensity of alpha/transparency, the change is visible in the main window. Press the Apply button to keep the change and close the dialog.

 

Note:A changed intensity can be saved ONLY the image has (or Color depth increased to) 32 BPP.

 

 

The other buttons can delete alpha/transparency and/or apply it to a color (and close the dialog).

 

 

Note:A 24 BPP image can keep/save the transparent color ONLY if saved as PNG.

 

 

To quit without changing the image, click Cancel.

 

This action can be reversed by the Edit Menu's Undo command.
