First/Last/Random File in Folder/List

 

 Top  Previous  Next

 

 

Click on the View Menu, then First File in Folder/List. The viewer displays the first supported file in the current folder/list.

 

Click on the View Menu, then Last File in Folder/List. The viewer displays the final supported file in the current folder/list.

 

Click on the View Menu, then Random File in Folder/List. The viewer displays a random supported file from the current folder/list.

 

 

Note:The files are sorted as set in menu Options->Sort list files.
