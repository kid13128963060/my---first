Insert overlay/watermark color/image

 

 Top  Previous  Next

 

 

Click on the Edit Menu, then Insert overlay/watermark color/image. A dialog allows you to insert an overlay color or image on the loaded image.

 

This feature can also be used in the Batch mode (Batch dialog, Advanced options, Add watermark image).

 

 

You can choose the start corner of the loaded image that you wish to measure from and the horizontal and vertical offset from that corner in pixels (X and Y) to where you want to place the overlay, or you can place the overlay centrally over the loaded image by clicking the Center button.

 

If you click into image (main window), the program will set the mouse coordinates as the start corner and show you a preview of the result in the main window.

 

You can choose which image file should be used as overlay and set a level of transparency for it, so that it displays as a watermark (0 is opaque, 100 totally transparent).

 

You may also use the mouse button to outline a selection area on the image before opening the dialog. When you then choose a start corner in the dialog the overlay position is preset to start from the corresponding corner of the selection. The required insets on the loaded image, X and Y, are calculated automatically.

 

 

Hint: you can use as an overlay an image (such as PNG or GIF format) where an area is already selected to be displayed as transparent. In this case that area will be displayed as totally transparent with the remainder of the overlay shown at the transparency selected in the dialog.

 

 

Click the OK button to add the text to your image. To close the dialog without adding any overlay, click the Cancel button.

 

Add overlay/watermark may be reversed with the Edit Menu's Undo command.
