Paste

 

Click on the Edit Menu, then Paste (or use the Paste tool on the toolbar).

 

It will paste the image from the clipboard (directly) into new image.

 

If you have an edit outline (selection rectangle) around an area of the current image, an image stored on the Windows Clipboard as bitmap, metafile or text will be pasted into your outline. The clipboard image will be resized to fit the outline and the rest of the image will stay as it is. This feature is very useful for editing images or portions of them.

You can change the selection paste option in Properties->Browsing/Editing.

 

Hint: if you paste something into selection, the image is not applied until you either click into the image or use a second paste (menu or CTRL+V) operation into same selection. Until then, you can move (right mouse button or arrow keys) or resize (left mouse button on the edges) the whole selection (with the image in it).

 

Hint: after paste into selection, SHIFT + Click into selection will apply the selection but keep the selection rectangle.

 

If there is no edit outline, an image (bitmap, metafile or text) on the Windows Clipboard will replace the entire image in the viewer.

The temporary filename for the pasted image can be set in Properties->Browsing/Editing.

 

 

 

Paste Special

 

Click on the Edit Menu, then Paste Special.

 

This option will add the clipboard image to the existing image. There are 4 menus for 4 directions, so you can create a simple Collage image.

You can change the fit option in Properties->Browsing/Editing.
