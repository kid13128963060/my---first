Canvas size / Add border/frame

 

 Top  Previous  Next

 

 

Click on the Image Menu, then Canvas size or Add border/frame.

 

Canvas size: a dialog allows you to choose the size of simple image border around the image and the border color.

Negative size values can be used to draw the frame on the inside of the image.

 

 

Add border/frame: a dialog allows you to choose the sizes and border colors of image border/frame around the image. The dialog offers several predefined frame effects and values. You can set up to 4 parts, up to 4 sizes/colors for every frame.

If a size value of a part is set to 0, that part won't be used.

Note:The effect can be used on a smaller portion (rectangle selection) of an image, as well as the entire image.

 

 

Note:The features are only available, if the PlugIn Effects is installed, which is done at program installation.

 

Click OK to create the new image.

 

This action can be reversed by the Edit Menu's Undo command.
