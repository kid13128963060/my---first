There are several options for loading of DigiCam RAW and other formats PlugIns.

If you want to use Canon DLLs for CRW/CR2 loading, please read the "i_plugins.txt" file for actual infos and links (IrfanView 32 bit ONLY).

 

Default text for TTF files allows you to specify the text, which is used for displaying true type font files. Enter the text, which is best for you to compare TTFs and to check the supported characters of the font.

 

Custom path for Adobe 8BF PlugIns allows you to specify the search path which will be used to show Adobe 8BF filters/effects in the menu Image->Adobe 8BF PlugIns (third submenu group). The default search folder is the IrfanView "PlugIns\Adobe 8BF\" folder, which is used if the edit field (custom path) is empty.
