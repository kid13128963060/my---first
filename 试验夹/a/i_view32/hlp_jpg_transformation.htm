JPG Lossless Rotation

 

 Top  Previous  Next

 

 

Note:This feature is available only if you have loaded a JPG file.

 

 

Click on the Options Menu, then JPG Lossless Rotation. A dialog allows you to rotate, flip, optimize or clean out one or many JPG files.

 

Note: The rotation dialog will set the EXIF orientation tag (if available) alwas to 1 (= top left; means: no rotation needed) !

 

You can process many JPG files (kind of batch mode), if you open the Thumbnails window and select several JPGs before you start this dialog.

 

DPI (dots per inch) data is stored either in the EXIF information or in the JFIF marker. When setting the images DPI, it will be written to the EXIF data if the EXIF data is present (and not cleaned). If the JFIF marker is activated, this will be updated/created too. If the image has neither EXIF data nor a JFIF marker, then a JFIF marker will be written to the file, the dialog will activate the JFIF option and play the system sound warning.

 

Lossless JPG operations like flip or rotate work only if the image height is a multiple of 16 (JPG specification). If this is not the case, unused image lines will be removed.

 

 

Note:Your original JPG file will be changed/overwritten!

 

 

Click Start to transform the loaded JPG or Cancel to exit.
