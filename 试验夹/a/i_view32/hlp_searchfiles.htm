Search Files

 

 Top  Previous  Next

 

 

Click on the File Menu, select Search Files. A dialog with search options opens. You can set filename options, start folder, file dates or text search.

 

Filename pattern examples:

 

*-        search all files, all extensions
*.*-        same as *
*.jpg-        find only JPG files
123*.jpg-        find only JPGs with 123 at name begin
*123.jpg-        find only JPGs with 123 anywhere in the name
image-        find only files which contain text "image" in the name

 

 

Text:: you can search for text in EXIF data (JPGs, TIFs, RAW-formats) and/or IPTC/Comment data (JPGs).

If several words are inserted (separated with space), all files with ANY of the words will be in the result list (like word_1 OR word_2).

 

Hint: if you just want to find a specific text with several words, you should set the complete text/phrase under "". Example: if you write the text as:

"our trip to Las Vegas"

(with "") you will get only files as result which contains the complete text/phrase:our trip to las vegas. But if you write it like:

our trip to Las Vegas

(without "") you will get all files which contains the words "our" OR "trip" OR "to" OR "las" OR "vegas" in any of the used metadata fields.

 

Hint: if you just want to check the existence of EXIF and/or IPTC and/or Comment data in a file, write as text a single * character.

 

Hint: if you want to exclude hidden folders/files from the search, disable the option for hidden folders/files in: Properties->Browsing->Show hidden files/folders.
