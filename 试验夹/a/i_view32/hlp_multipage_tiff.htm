Multipage images

 

 Top  Previous  Next

 

 

Click on the View Menu, point at Multipage images. A sub-menu opens with several options to click.

 

The options are:

 

Note:Following menus are activate only, if you have an image with many pages (some DCX/TIF/ICO/JPM/PDF files) loaded.

 

Next page

Click Next Page (or use the Next Page tool on the toolbar). The viewer displays the next page in sequence.

 

Previous page

Click Previous Page (or use the Previous Page tool on the toolbar). The viewer displays the previous page in reverse sequence.

 

First page

Click First Page. The viewer displays the beginning page of the image.

 

Last page

Click Last Page. The viewer displays the final page of the image.

 

Open page number ...

Click this menu and a dialog will be displayed. You can enter a page number to open.

 

Show page thumbnails ...

Click this menu and a dialog with page thumbnails will be displayed. Use double click or space/enter to load a page in the main window.

Use TAB to switch focus to main window. The dialog uses the thumb settings from the Thumbnails window.

 

Auto show page thumbnails dialog

Click this menu to enable the auto-start of page thumbnails dialog when a multipage image is loaded. Click again to disable the option.

 

Start animation

Click this menu and choose the time intervall between the pages. The animation will be started.

 

Stop animation

Click this menu to stop the animation.
