Remove/Insert Strip

 

 Top  Previous  Next

 

 

Use the left mouse button to outline the area (selection rectangle), click on the Edit Menu, then Remove/Insert Strip.

 

You can choose to remove or insert a vertical or horizontal strip. The strip width or height is based on the width OR height of the selection.

Vertical strip uses the selection width, horizontal strip uses the selection height and the other dimension is the image height (vertical strip) or image width (horizontal strip).

 

Insert strip uses the background color from the Cut operation, which can be set in Properties->Browsing/Editing.

 

After the operation, the selection stays in the image, so you can add more strips or remove previously inserted strip(s).

 

Remove/Insert Strip may be reversed with the Edit Menu's Undo command.
