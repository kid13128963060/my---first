Create new JPG EXIF Data

 

 Top  Previous  Next

 

 

Note:This feature is available only, if you have loaded a JPG file and if the JPG file does NOT contain EXIF data.

 

 

Click on the Options Menu, then Create new JPG EXIF data. A dialog allows you to write the most important EXIF tags.

 

If the JPG file already contains EXIF data, you will see an error text.

 

Note:You can't change existing EXIF data. You can delete the EXIF data from a JPG (take care) in the menu JPG-Lossless Operations and then add fresh EXIF data.
