Replace color

 

 Top  Previous  Next

 

 

Click on the Image Menu, then Replace Color. A dialog appears with the options. You can choose which color should be replaced with another color.

 

With a click into the main window you can set which color should be replaced (source color).

 

Note:The effect can be used on a smaller portion (rectangle selection) of an image, as well as the entire image.

 

When you have finished using the dialog, click Exit.
