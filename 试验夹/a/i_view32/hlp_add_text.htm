Insert text

 

 Top  Previous  Next

 

 

You can start the dialog and place the text using mouse click into image or you can use the left mouse button to outline the area (selection) on the image where you want text to appear.

 

How to make a selection? Just press the left mouse button and move the mouse. You will see the selection rectangle. If you click on the image again outside the selection, the selection rectangle will be deleted and the selection is no more active.

 

Click the Edit Menu, then Insert text. A dialog box will open.

 

If this feature is used in the Batch mode (Batch dialog, Advanced options, Add overlay text), the fields for coordinates are enabled and you can set the start corner and size of the rectangle which will be used for text.

 

 

In the Text box on the left, type in the text you want to put on the image. The text can use placeholders.

 

Use the Choose Font button to change the size, style, color, etc. of the text. The name of the selected font is shown below the text box.

NOTE: The font size is the result text size in pixels ... for Windows display mode of 100%. If you are using text/DPI scaling in system display options, the font size will be also scaled.

         You should test this to see which font size is the "best" for your images.

 

 

If the displayed image is zoomed-in (zoom size < 100%) you can use the option Adjust font size to current zoom size to make the text "correct" for the current display/zoom. Example: the current image zoom size is 66%, the selected font size is 12 points. If the adjust option is used, the inserted font size will be 18 points (=font x 100/zoom), which look like 12 points in the current zoom.

In Batch mode is the desktop height used to calculate zoom.

 

Uncheck the Background is transparent box to choose a background color. The highlighted area will change to that color, with the text inside. The small box left the button shows the selected background color.

 

The option for Text rotation allows you to rotate text. E.g. 90 or -90 deg will draw the text vertically. When you change the deg value, click in the image (main window) to see the preview.

 

The option Insert mode can change the text position. Default is to insert the text into image (e.g. choose position by clicking into the main window/image). If you choose to add the text above or below the image, the program will add a Canvas area into the image and place the text there (like if you use Image->Add Canvas first).

 

 

The dialog offers several combinations of colors and effects, feel free to play with the combinations.

 

You can save the current/visible dialog values as a profile and later use/load the saved values.

 

Click the Preview button to see (main window) what will happen. You can change options and use preview many times, until you have a satisfactory result.

 

Click the OK button to add the text to your image. To close the dialog without adding any text, click the Cancel button.

 

Add text may be reversed with the Edit Menu's Undo command.

 

Hint: If you use CTRL key + click into image (main window), the text will be added at the click position and you can add a new text.

 

Hint: If you use SHIFT key + click into image (main window), the color from the image will be set as text color.

 

Hint: the dialog will remember the last screen position.
