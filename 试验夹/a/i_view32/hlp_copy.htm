Copy

 

 Top  Previous  Next

 

 

Use the left mouse button to outline the area (selection), click on the Edit Menu, select Copy (or use the Copy tool on the toolbar).

 

The area inside the outline will be placed on the Windows Clipboard as bitmap image and may be pasted in elsewhere.

 

If you don't select an area with the mouse, the entire image will be copied to the Windows Clipboard.
