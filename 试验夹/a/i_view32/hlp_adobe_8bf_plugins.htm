Adobe 8BF PlugIns

 

 Top  Previous  Next

 

 

Click on the Image Menu and click Adobe 8BF PlugIns.

 

 

Filter dialog will open a dialog from which you can load and perform Adobe Photoshop compatible PlugIns/filters/effects. Use the button Add 8BF Filters to select/import a folder with 8BF filters. Select one of the loaded/shown filters and start it using Space key or button or double click.

 

 

Second submenu group: shows you some very useful Adobe Photoshop compatible PlugIns/Effects for faster start. These standard effects can be downloaded and installed from IrfanView website:

 http://www.irfanview.net/plugins/irfanview_adobe_8bf_plugins.exe

or as ZIP:

 http://www.irfanview.net/plugins/irfanview_adobe_8bf_plugins.zip

 

 

Third submenu group: you can add additional 8BF PlugIns/filters to these menus. You can either copy the 8BF files (and any required DLLs) to the IrfanView folder Plugins\Adobe 8BF\ or you set the search location for 8BF files in Properties->PlugIns.

 

 

Hint: Some nice sites with 8BF effects are:

 http://www.richardrosenman.com/software/downloads/       (like Grid Generator etc.)

 

 

The applied effect after pressing OK can be reversed by the Edit Menu's Undo command.
