Copy Shop

 

 Top  Previous  Next

 

 

On the File Menu, click Copy Shop.

 

A dialog appears with options for printing an image directly from a scanner or any other TWAIN compatible device (digital camera, TV card, etc.).

 

The button Select TWAIN Source lets you choose a TWAIN compatible device (usually a scanner) from a list.

 

Check option Show TWAIN preview dialog, if you want to see a preview before the scanned image is sent to the printer.

 

Enter the DPI value for the scan; default is 100. Usually, half of the selected printing resolution is a good value for scanning.

 

Select the active printer and set the printer properties.

 

Enter the Number of copies for printing.

 

Note:For printing the actual settings from the print dialog are used.

 

When everything is set, click Start. To exit without making a selection and copying, click Cancel.
