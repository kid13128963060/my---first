Resize/Resample

 

 Top  Previous  Next

 

 

Click on the Image Menu, then Resize/Resample. A dialog allows you to choose the new size for the image; some Resample filter can be used if you are enlarging the image.

 

There are several built in choices, or you can type over the old numbers. The first number is width, the second height, both in pixels.

Leave the Preserve aspect ratio box checked, type in one number and the other will be calculated for you.

 

Resizing an image with Resampling an image often improves its quality, especially when enlarging it. Resizing without resampling can result in a blurry image.

 

You can set if the Resample filters are used for enlarging/shrinking or if a special/fast resample filter should be used for shrinking only. Normal resample filters are slower than the fast filter for shrinking.

 

Resample filters/operations can slightly change image gamma/brightness, so you can activate an option which can improve gamma for resampled images (for experts).

 

Important note: The option "Try to improve gamma for Resample" will (slightly) change the input (original) image (even if only for display in the main window)! Do not use this option if exact colors are required for further image editing.

 

Hint: if you set the units to cm or inches you can resize the image only by changing the DPI value.

Example: the image is 600x500 pixels and 300 DPI, if you set unit to cm/inches and new DPI to 150, new image will be 300x250 pixels, 150 DPI.

Same option is available in Advanced-Batch dialog (Resize based only on DPI value).

 

 

Click OK to resize the image.

 

This action can be reversed by the Edit Menu's Undo command.
