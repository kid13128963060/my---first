Supported File Formats

 

 Top  Previous  Next

 

 

These are the file formats/extensions supported by the current version of IrfanView:

 

AIF¹, ANI, AU¹, AVI, AWD, B3D, BMP, CAM, CLP, CPT, CRW/CR2, CUR, DAT¹, DCM/ACR, DCX, DDS, DIB, DJVU, DLL/EXE, DPX/CIN, DXF/DWG/HPGL/CGM/SVG, ECW, EMF, EPS/PS/PDF/AI, EXR, FITS, FLI/FLC², FLIF, FPX, G3, GIF, GLCD, HDP/WDP, HDR, HEIC, ICL, ICO, IFF/LBM, IMG, JLS, JP2, JPG/JPEG, JPM, JXL, KDC, Mac PICT², MAG, MED, MID, MNG/JNG, MOV¹², MP3, MPG/MPEG¹, MPO, MrSID, OGG, PBM, PCX, PDF, PDN, PGM, PhotoCD, PNG, PPM, PSD, PSP, PVR, QOI, QTIF², RAS/SUN, RealAudio, RAW, RLE, RMI, SFF, SFW, SGI/RGB, MrSID, SWF/FLV, SND¹, SVG, TGA, TIF/TIFF, TTF, TXT, WAD, WAL, WAV, WBMP, WMF, WMA, WMV, WBC/WBZ, WEBP, WSQ, XBM, XCF, XPM, (DNG, EEF, NEF, ORF, RAF, MRW, DCR, PEF, SRF, RW2, NRW).

 

Many old formats (Amiga, Atari, C64, ZX Spectrum etc.) are also supported using the RECOIL library.

 

Note:Many of these formats are supported through PlugIns, check the table below and read the file "i_plugins.txt" for more infos.

 

PlugIns Download:        http://www.irfanview.com/plugins.htm
