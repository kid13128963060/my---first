Properties, Viewing

Handle alpha/transparent color: Some images (like PNG/TIF/TGA/GIF/WEBP/DDS/SVG etc.) can be saved with transparency color or alpha channel. The image areas defined with alpha/transparecy can be shown with this option with a checkerboard image, the color of the main window of IrfanView (can be set below, Main window color) or the transparency will be discarded. If you keep the transparency it will be always used, e.g. if you edit the image or save it.

 

Show embedded TIF Annotations: For TIFs created using Wang/Kodak Imaging: the saved text annotations can be shown in the image.

 

Dither 24 BPP images on 16 BPP displays: Click this, if you want better displaying of the true color images, if you have 65536 colors under your Windows.

 

Apply gamma correction: Enable it, if you want to change the intensity of each loded image. The default value is 1.00 (no change), the range is between 0.01 (extreme dark) and 6.99 (extrem bright). Use this option for viewing a photo series with to low or to high exposure. This option is especially useful on notebook computers, which have been set to a lower display brightness for saving power in battery mode. With an appropriate value above 1.00 for the gamma correction, the images can be viewed nevertheless with their real brightness.

 

Gamma correction for BW images is activated by default and is used for setting the intensity of resampled black/white images. The default value is 0.50 (darker), the range is between 0.01 and 6.99.

 

Show full file path in the title bar: Check it, if you want to see the full file path (incl. folders) in the main window title bar. Otherwise only the short name (filename + extension) is shown (default).

 

Use right mouse button for context menu: The right mouse button can be used for scrolling (default) or to show the context menu.

 

Show mouse coordinates in the Statusbar: if you want to always see the mouse position in the image.

 

Keep mouse on Next/Previous-file toolbar buttons: if you use mouse to click on these toolbar buttons, you can force mouse cursor to stay on these buttons after the new image is loaded.

Useful if the center-window option is used (window moved/resized for the new image).

 

Enable Zoom Magnifier: this option can enable/disable the magnifier window, which can be used for additional zooming of the underlying area. Possible also in fullscreen mode.

CTRL + SHIFT + Mouse move: Show magnifier window - mouse wheel zooms within magnifier area, click inside closes it.

 

The option Enable Dark mode will paint IrfanView and most dialogs/controls with dark colors and white text.

 

The next option is Center window when loading new image. Click this and each time you load a new image in the viewer, it will be centered on your desktop.

 

The next option is Remember the last window size/position. This option works only if you use the following viewing options (menu: View->Display options): "Fit images to window" or "Do not fit anything". Click this and each time you open the viewer, it will be positioned on your screen where it was when you last closed it.

 

Main window color lets you define the background color of the main window of IrfanView. You can choose any color as background. Default is black.

 

Status bar text: you can set the text displayed in the status bar field on the right. Check this page for more infos about text possibilities.
