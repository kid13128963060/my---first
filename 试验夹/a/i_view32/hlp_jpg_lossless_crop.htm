JPG Lossless Crop

 

 Top  Previous  Next

 

 

Note:This feature is available only, if you have loaded a JPG file and if a rectangle selection is created/shown in the image.

 

 

Click on the Options Menu, then JPG Lossless Crop. A dialog allows you to set where the result crop file will be saved.

 

Note: You can process many JPG files, if you open the Thumbnails window and select several JPGs before you start this dialog.

 

 

Note: Lossless JPG crop works only if the start coordinates (X and Y) are set on JPG block sizes. The block sizes can be 8x8, 16x8, 8x16 or 16x16 pixels, depending on the how the JPG was compressed. If the option Align X/Y coordinates is ON, the whole crop rectangle will be moved to the closest block, otherwise only X and/or  Y coordinate will be moved to the left or top of the current block.

Example: crop rectangle is set to: X = 43, Y = 32, width = 100 and height = 100; JPG block size is 16x16 pixels.

If the option Align X/Y is set to ON, the X coordinate will be moved to 48 (nearest block is on the right) and the result image will be 100x100 pixels.

If the option Align X/Y is set to OFF, the X coordinate will be moved to 32 (always to the left, to the start of the current block) and the result image will be 111x100 pixels (111 = (43-32) + 100).

 

 

Click Start to crop the loaded JPG or Cancel to exit.
