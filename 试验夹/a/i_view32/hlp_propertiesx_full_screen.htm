Properties, Fullscreen/Slideshow

 

 Top  Previous  Next

 

 

1 Show images/movies with the original size: At this option all images and videos are always displayed on screen without size adaption. Images/movies are automatically centered, if they are smaller than the screen. At too large images/movies only the upper left area of the image, which fits on screen, can be viewed. You can change the displayed area of the image/movie with the arrow keys on the keyboard to view also other areas.

 

2 Fit to screen: large images only: This is the recommended display option for the full screen mode. Images, which are too large for the current screen resolution, are reduced in size to fit fully to screen without changing the aspect ratio (proportional). Small images are displayed centered in original size as at first option.

 

3 Fit to screen: all images/movies: Is similar to second option, but too small images/movies are enlarged proportional to full screen size.

 

4 Stretch all images/movies to screen: At this option all images/movies are adapted in size to fill out the complete screen. The aspect ratio of the images/movies is not taken into consideration at size adjustment, which can result in a strong distortion of the image.

 

5 Fit image width to screen width: The image width will be fitted/stretched to screen width, height is proportional.

 

6 Fit image height to screen height: The image height will be fitted/stretched to screen height, width is proportional. Useful for panorama images.

 

7 Fit small image side to screen: The smaller image side (width or height) will be fitted/stretched to screen, the other image side will be proportional and can be probably scrolled.

 

At full screen mode you can always switch temporary to one of these 7 display options with the keys 1, 2, 3, 4, 5, 6 and 7. Changing the display option during full screen mode with the keys does not change the saved display option set in this dialog.

 

 

Set display multiplier: this option is useful for wide screen displays, to set a zoom factor for width/height.

 

Center image on screen: this option can be used to display the image in the center of the fullscreen or in the top left corner. You can center the image in several ways: vertical or horizontal or both.

 

Use "Resample" function for first display of an image: Activate this option to get the best possible version of an image in full screen mode (initial display of the image), if resizing the image is necessary. But the use of resample for size adjustment needs more calculating (loading) time.

 

Use "Resample" function for zooming: Activate this option to get the best possible version of an image in full screen mode, if resizing/zooming the image is necessary. But the use of resample for size adjustment needs more calculating (loading) time.

 

Show vertical blurred image sides: Activate this option to show blurred parts of the visible image if the image height is same or larger than screen height and there is empty screen/side area (left and right side) to be filled with. The center-image option must be also ON for this feature.

 

Screen color allows the definition of the screen color during loading the images and the remaining area between image border and screen border depending on the current setting of the full screen display option.

 

With Text font you can set the text font with its text style, size and color, which will be used for displaying the text. The also adjustable Text background color is visible, if the option Show transparent text is not active. In this case the text is displayed in a box with the selected text background color, which guarantees an always good readable display of the text. At transparent text display only the text overlays the image, which has a less annoying effect, but the text can be poor readable depending on the image.

 

If option Multiple monitor mode is enabled, the images are displayed stretched over all connected monitors forming a big screen.

 

 

The last two options are only for full screen mode.

 

The checked option Hide mouse cursor prevents the mouse cursor from showing on screen while the images are displayed.

Even if the option is set and mouse hidden, if you move the mouse, it will be shown for a short time and you can use it.

 

Click the option Show text to display the file infos as text as the image appears on screen. IrfanView can display a variety of file infos like name, size, date/time, EXIF/IPTC/Comment infos, etc. Check this page for more infos about text possibilities.
