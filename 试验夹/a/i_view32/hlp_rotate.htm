Rotate Left

Click on the Image Menu, then Rotate Left. The image will turn 90 degrees; the top of the image will move towards the left.

 

 

Rotate Right

Click on the Image Menu, then Rotate Right. The image will turn 90 degrees; the top of the image will move towards the right.

 

Hint: for JPG images, you should better use the lossless transformation in Options->JPG Lossless Operations menu.

 

 

Custom/Fine rotation

A dialog allows you to enter a special degree (range: -360 - +360) for the image rotation.

Enter a value to control how far the image is rotated; positive numbers (e.g. 30) rotate the image to the right, negative (e.g. -30) rotate the image to the left; higher numbers rotate the image farther; default is 45. You can also set the color of the background area, produced by the rotation.

 

Note:The effect can be used on a smaller portion (rectangle selection) of an image, as well as the entire image.

 

Note:You can use the Paint PlugIn (Edit menu of F12 hotkey) to draw a line to straighten the image.

 

Note:Custom/Fine rotation is only available, if the PlugIn Effects is installed, which is done at program installation.
