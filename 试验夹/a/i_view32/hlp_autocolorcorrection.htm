Auto color correction

 

 Top  Previous  Next

 

 

Click on the Image Menu and click Auto color correction.

 

The feature improves the brightness of the image using automatic calculations for brightness and gamma values.

 

Note:You can improve the result if you first create a selection/outline in the image (important area) using the left mouse button. Hold the CTRL key while you click the menu. In this case, only the selected area will be used for calculations but applied to whole image, otherwise the feature will be applied on the selection only.

 

This action can be reversed with the Edit Menu's Undo command.
