JPM

 

Set the profile option. There are 5 standard profiles for different image/color types. You can also set the quality of the result JPM image.

 

If you want the thumbnail in the JPM image (for faster preview), check Save thumbnail option.

 

If you want to create multipage JPM images, click on the Options menu, then click Multipage images -> Create Multipage JPM.

 

Note:Saving as JPM is only available, if the PlugIn JPM is installed.

 

 

After setting the options you choose, click Save to save the image and store the settings. To exit without saving the file or changed options, click Cancel.

 

The new settings will stay in effect until you change them again.
