Multipage images

 

 Top  Previous  Next

 

 

Click on the Options Menu, point at Multipage images. A sub-menu opens with several options to click.

 

The options are:

 

Create Multipage TIF

Check this option to create multipage TIF images from single or multipage images. You can save each page with a different TIF compression. If you want to set same compression for all pages, set the compression before you add any image.

 

Create Multipage PDF

Check this option to create multipage PDF images from single or multipage images.

 

Note:PDF format is used as PlugIn, download and install:

http://www.irfanview.com/plugins.htm

 

Edit Multipage TIF/PDF

Check this option to edit (arrange, delete or add) pages/images in the opened multipage TIF/PDF file. The changes can be applied to the original file or a new TIF/PDF file is created (recommended).

 

Append current image to TIF/PDF

Check this option to append the displayed image to a TIF/PDF file. The actual TIF/PDF save option (save dialog) will be used for the new page.

 

Print all pages

Click Print all pages. The print dialog may be used to send all pages of the image to your printer.

 

Extract all pages

Click Extract all pages. A dialog allows you to select the folder/directory, in which the pages will be stored.

 

The pages from the image are placed into the folder/directory with names like this:

 

 filename_irfanview_extract_001.bmp

 

where name is the name of the file and 000 is the number of the page. The pages are stored as bitmap (BMP) files.
