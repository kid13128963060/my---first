Zoom In/Out

 

 Top  Previous  Next

 

 

Click on the View Menu, then Zoom In or Zoom Out (or use the Zoom tools/buttons on the toolbar).

 

Zoom In increases size of the image. Click again for more increase.

 

Zoom Out decreases size of the image. Click again for more decrease.

 

The amount of each increase can be set using Zoom step on the Options Menu->Properties->Viewing tab. The default is 10% increase per use, calculated on the previous zoom value.

 

 

Note:Zoom changes only the display size, not the image size! If you want to change the image size, see the menu Image->Resize/Resample.
