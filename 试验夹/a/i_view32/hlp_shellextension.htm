IrfanView Shell Extension

 

 Top  Previous  Next

 

 

IrfanView Shell Extension is a small program which adds a Context menu to Windows shell, which can be used in any file manager like Windows Explorer, Total Commander etc..

 

You can find and download IrfanView Shell Extension on the PlugIns website:

 http://www.irfanview.com/plugins.htm

 

 

When installed, the shell context menu will get a new entry/menu "IrfanView operations" which allows you to call IrfanView with the selected files and perform different operations.

 

Available operations are:

 Play slideshow with selected files (using current IrfanView slideshow dialog settings)

 Open selected files in IrfanView Thumbnails window

 JPG Lossless Rotation (automatic rotation, based on EXIF orientation, optimize ON, keep all markers ON)

 Convert pictures/images to another format (same folder, copies will be created if the result file already exists)

 Save filenames as TXT file (same folder)

 Create multipage TIF or PDF (same folder)

 Create panorama image (and show it in IrfanView window)
